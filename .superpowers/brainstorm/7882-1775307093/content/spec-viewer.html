Global Market Hub — Design Spec
Restructure equity dashboard into a 6-market analytical hub
✓ Approved 2026-04-04 Analyst / Researcher
Goals
IN SCOPE
✓Single app to monitor 6 asset classes: equities, bonds, FX, derivatives, real estate, insurance
✓Each market workspace purpose-built for that asset class — not a generic template
✓Clean market switching with minimal friction (top tab bar)
✓Mock data first; swapping in real APIs is a one-file change per market
OUT OF SCOPE
✗Cross-market correlation alerts or signal engine
✗Real-time data for all markets on day one
✗Mobile layout (desktop-first)

Decisions Log
Navigation pattern
Top tab bar — markets as tabs, per-market sub-tabs below
Workspace structure
Tailored views per market — each asset class gets domain-specific visualizations
Data strategy
Mock data first; each mock shaped to match its real API's response format
Implementation approach
Market Router — src/markets/{name}/ per market, thin App.jsx shell
Build order
Hub Shell → FX → Bonds → Real Estate → Derivatives → Insurance
